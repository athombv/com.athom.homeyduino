Homeyduino

Den här Homey-appen låter dig enkelt koppla dina egna Arduino-baserade kreationer med din Homey via ethernet eller WiFi. Att koppla dina egna kreationer med Homey har aldrig varit enklare: installera bara appen och ladda ner Homeyduino Arduino-biblioteket för att komma igång!

För mer information om hur du gör detta och för att komma igång besök utvecklarens webbplats nedan.
