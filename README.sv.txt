Homeyduino

Den här Homey-appen gör det enkelt att ansluta dina egna Arduino-baserade skapelser till din Homey med hjälp av ethernet eller WiFi. Att ansluta dina egna skapelser till Homey har aldrig varit enklare: installera bara appen och ladda ner Homeyduino Arduino-biblioteket för att komma igång!

För mer information om hur du gör detta och kommer igång, besök utvecklarens webbplats nedan.
