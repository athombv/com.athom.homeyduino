Homeyduino

Cette application Homey vous permet de connecter facilement vos propres créations basées sur Arduino avec votre Homey en utilisant Ethernet ou WiFi. Connecter vos propres créations avec Homey n’a jamais été aussi simple : il vous suffit d’installer l’application et de télécharger la bibliothèque Arduino Homeyduino pour commencer !

Pour plus d’informations sur la manière de procéder et pour bien démarrer, visitez le site web du développeur ci-dessous.
