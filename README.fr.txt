Homeyduino

Cette application Homey vous permet de connecter facilement vos propres créations basées sur Arduino à votre Homey en utilisant l'ethernet ou le WiFi. Connecter vos propres créations à Homey n'a jamais été aussi simple : il vous suffit d'installer l'application et de télécharger la bibliothèque Arduino Homeyduino pour commencer !

Pour plus d'informations sur la façon de le faire et pour démarrer, visitez le site Web du développeur ci-dessous.
